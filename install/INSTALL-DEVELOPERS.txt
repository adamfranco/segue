Segue Developer Install instructions:

1) Obtain Segue, Harmoni, and Polyphony
	A) If you downloaded the segue bundle, unzip it to a web accessible directory
	
	B)	a) Check out from CVS or download the following packages
			- Segue (this package that contains this file):
				http://sf.net/projects/segue/
				CVS: http://sourceforge.net/cvs/?group_id=88266
				Files: http://sourceforge.net/project/showfiles.php?group_id=88266
			- The Harmoni Framework
				http://sf.net/projects/segue/
				CVS: http://sourceforge.net/cvs/?group_id=82873
				Files: http://sourceforge.net/project/showfiles.php?group_id=82873
			- Polyphony
				(Part of the Harmoni project)
			- DOMIT! XML parser
				http://www.engageinteractive.com/domit/
				
		b) Place Segue, Harmoni, Polyphony, and DOMIT! in the same web accessible 
		directory
	
2) Create a MySQL database for Segue to use.
	A) create the database
	B) give a database user full permission to that database.


3) Create your database config
	A) copy 
			segue/config/database_default.conf.php
		to
			segue/config/database.conf.php

	B) Modify your database config
		a) change the database connection entry in the config to reflect the database that
			you just created.
		b) comment-in and modify the LDAP authentication piece in
				segue/config/authentication.conf
			if you wish to use that.


4) Go to Segue. 
	The installer script will automatically run for the first page load of each
	browser session. If you need to force the installer script to run again,
	clear your browser's Segue session cookie and refresh the segue page.


6) If desired, load the sample collection:
	A) In Segue, go to 
			AdminTools --> Import
		and select "XML" from the import-type menu.
	B) Click the Browse... button and select the following file:
		segue/sampledata/SampleCollection.xml
	C) Click the "Import".
		The import may take several minutes due to the need to download the sample
		data from SourceForge.
	D) Once the import has finished, click "Cancel" to exit the Import wizard
		and return to Segue. You should now see "Sample Media Collection" under
		"Collections".


7) To reset the Segue databases to their freshly-installed state, deleting all data,
	go to:
		AdminTools --> RefreshSegue
